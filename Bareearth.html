Bare Earth and Feature Extractions

Processing LiDAR datasets to generate bare earth models and extract terrain and surface features for spatial analysis.

Project Overview

This project focuses on processing LiDAR point cloud data from the National Map to create a bare earth digital terrain model (DTM) in Coralville by separating ground returns from above-ground features.

Feature extraction techniques were applied to identify buildings, vegetation, and contours, supporting future spatial analyses and figure generation for subsequent assignments.
