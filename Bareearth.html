Bare Earth and Feature Extractions

Processing LiDAR datasets to generate bare earth models and extract terrain and surface features for spatial analysis.

Project Overview

This project focuses on processing LiDAR point cloud data to create a bare earth digital elevation model (DEM) by removing surface objects such as buildings and vegetation.

Feature extraction techniques were applied to identify key landscape elements, including elevation contours, vegetation layers, and built structures, supporting terrain analysis and environmental modeling.
