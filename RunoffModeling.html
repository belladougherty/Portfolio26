Flood Analysis of Cedar Rapids

Evaluating flood hazard in downtown Cedar Rapids using LiDAR-derived elevation data and flow modeling.

Project Overview

This project analyzes flood hazard in Cedar Rapids, Iowa, particularly downtown along the Cedar River, using LiDAR-derived elevation data combined with relevant geospatial datasets.

Flow modeling identifies low-lying areas susceptible to flooding by examining topography, slope, and impervious surfaces, assessing how these factors influence runoff accumulation and concentration.
