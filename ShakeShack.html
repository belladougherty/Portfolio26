Shake Shack’s Network Analysis in NYC

Network analysis of Shake Shack locations in New York City, assessing which stores should close based on service areas.

Project Overview

In this project, I performed a network analysis of Shake Shack locations in New York City, assessing the impact of location factors such as service area coverage.

Tools used in the analysis include ArcGIS and QGIS for spatial analysis and mapping. The project aimed to provide actionable insights for better network optimization and decision-making.
