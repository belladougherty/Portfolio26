Bigfoot Spatial Analysis

Spatial analysis of reported Bigfoot sightings to identify clustering patterns and relationships with terrain, elevation, forest cover, and soil type.

Project Overview

This project examines reported Bigfoot sightings across the United States using spatial analysis techniques to identify geographic clustering and environmental patterns.

Kernel density estimation and terrain analysis were used to evaluate whether sightings correlate with forested regions, elevation, landscape features, and soil type. Soil type data is available in raster format from the US Department of Agriculture's Natural Resources Conservation Service (NRCS). Incorporating soil type into the suitability model would involve normalizing the raster, assigning weights based on importance, combining it with the other criteria using a weighted overlay approach, and calculating a final suitability score for each cell.
